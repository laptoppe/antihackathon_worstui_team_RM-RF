HOW TO PLAY:

Click DEAL LETTERS to send LETTERS to ROWS.

Click PICK next to ROW to take rightmost LETTER into HAND.
Click PICK many time on same ROW to take MORE LETTER but only if VALID STACK

Click SET next to ROW to place LETTER from HAND but only if VALID STACK.

USERNAME FIELD does not follow STACK RULE.
Click SEND to submit USERNAME.

STACK RULE:

STACK is important rule.
STACK is string of LETTER.
STACK is VALID if all LETTER in STACK follow STACK ORDER.
STACK ORDER is as follows:

[left] (0123...9) > (ABCD...Z) > (abcd...z) [right]

SETting LETTER from HAND to ROW only requires that RIGHT LETTER of ROW and LEFT LETTER of HAND STACK be VALID STACK.
If RIGHT LETTER of ROW and LEFT LETTER of HAND is not VALID STACK, cannot SET.
ROW does not need to be VALID STACK. DEAL LETTERS does not necessarily make VALID STACK
HAND can only hold VALID STACK. HAND does not accept LETTER out of ORDER.
